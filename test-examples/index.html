three-edit Examples

This page contains examples for each major module of the three-edit library.

Primitives

  • Primitives Examples

    Examples of creating basic 3D primitives like planes, cubes, spheres, cones, cylinders, and arrows.

  • Primitives Bug Examples

    Examples demonstrating known bugs in primitives.

    Note: Contains examples of known bugs in createCone and createArrow.

  • Fixed Primitives Examples

    Examples using fixed implementations of primitives.

    Note: Contains fixed implementations of createCone and createArrow.

Core

  • Core Basics Examples

    Basic examples of core functionality like EditableMesh, Vertex, Edge, and Face.

  • Core Advanced Examples

    Advanced examples of core functionality like topology operations, history/undo, and mesh queries.

Editing

  • Editing Examples

    Examples of mesh editing operations like extrude, bevel, inset, loop cut, and bridge.

Transform

  • Transform Examples

    Examples of transform operations like translate, rotate, scale, and mirror.

Selection

  • Selection Examples

    Examples of selection utilities like vertex selection, edge selection, face selection, edge loop selection, and edge ring selection.

UV Mapping

  • UV Mapping Examples

    Examples of UV mapping operations like planar projection, box projection, cylindrical projection, spherical projection, and smart unwrap.

IO (Import/Export)

  • IO Examples

    Examples of importing and exporting 3D models in OBJ and GLTF formats.

All-in-One Examples

  • Direct Test

    A simple test page that loads and tests the three-edit library directly.

  • Simple Test

    A minimal test page that loads Three.js primitives for comparison.
